## Abstract

We introduce a one‑parameter family of generalized logistic maps that incorporate a memory term and a feedback threshold. The recurrence

```
x_{n+1} = x_n + i·(x_n − x_{n−1})·k·(1 − x_n)
```

with a threshold condition that reduces k when x exceeds a critical value (e.g., 1.5) produces a rich variety of periodic, chaotic, and stabilized orbits. By extending the system to two dimensions with rotations by multiples of π/5 and scaling factors of 1/φ, where φ = (1+√5)/2 is the golden ratio, we construct strange attractors with pentagonal symmetry. The recurrence naturally connects to Fibonacci numbers and the self‑similar geometry of quasicrystals.

## 1. Introduction

The logistic map x_{n+1} = r·x_n·(1−x_n) is one of the simplest models of nonlinear dynamics, exhibiting period‑doubling bifurcations and chaos for r > 3.57. Its behaviour is governed by a single parameter r. In this paper we propose a natural generalisation that includes a memory term and a feedback mechanism.

The golden ratio φ = (1+√5)/2 ≈ 1.618 appears ubiquitously in systems with pentagonal symmetry, from quasicrystals to phyllotaxis. We show that by choosing rotation angles as multiples of π/5 and scaling factors as 1/φ, the two‑dimensional extension of the generalised map generates strange attractors with striking pentagonal patterns.

---

## 2. A Generalized Logistic Map with Memory

Define the recurrence:

```
x_{n+1} = x_n + i·(x_n − x_{n−1})·k·(1 − x_n)
```

where:
- i is an **intensity** parameter (how much the previous increment influences the next step)
- k is a **tolerance** parameter (a damping factor)
- The term (1 − x_n) ensures boundedness in [0,1] when i and k are small

**Feedback control:** When x_n exceeds a threshold T (e.g., T = 1.5), we reduce k to a smaller value (e.g., k = 0.8). This prevents the orbit from diverging and introduces a stabilising mechanism.

**Special case:** With i = 1 and k = 1, and ignoring the threshold, the map becomes:

```
x_{n+1} = x_n + (x_n − x_{n−1})·(1 − x_n)
```

which is reminiscent of the Fibonacci recurrence x_{n+1} = x_n + x_{n−1} when x_n is small.

## 3. Two‑Dimensional Extension with Pentagonal Symmetry

Let z = x + iy be a complex number. Define:

```
z_{n+1} = s·R·z_n + t
```

where:
- s = 1/φ ≈ 0.618 is the scaling factor
- R is a rotation by θ = π/5 (72°) or multiples thereof
- t is a translation vector (e.g., t = 1 + 0i)

Choosing two maps (as in an iterated function system) with different rotations produces an attractor with pentagonal symmetry. The natural rotation angles are multiples of π/5 because φ appears in the relation cos(π/5) = φ/2.

## 5. Connection with Fibonacci Numbers

The Fibonacci sequence F_n satisfies F_{n+1} = F_n + F_{n−1}. This recurrence is a special case of our generalised logistic map when x_n is small, i = 1, k = 1, and the (1−x_n) factor is omitted. The ratio F_{n+1}/F_n → φ as n → ∞. Thus the map naturally encodes the golden ratio.

Moreover, the appearance of pentagonal symmetry in the two‑dimensional attractor is intimately linked to φ through the relation:

```
1/φ = φ − 1,   and   φ = 1 + 1/φ
```

and the identity 1/φ + 1/φ² = 1, which underlies the self‑similarity of many golden‑ratio‑based fractals.

## 6. Numerical Experiments

A simple Python implementation of the two‑dimensional system:

```python
import numpy as np
import matplotlib.pyplot as plt

phi = (1 + np.sqrt(5)) / 2
s = 1 / phi
theta = np.pi / 5
R = np.exp(1j * theta)

z = 0 + 0j
points = []
for _ in range(10000):
    if np.random.random() < 0.5:
        z = s * z
    else:
        z = s * R * z + 1
    points.append(z)

x = [p.real for p in points]
y = [p.imag for p in points]
plt.scatter(x, y, s=0.1, c='black')
plt.axis('equal')
plt.title('Pentagonal Strange Attractor')
plt.show()
```

The resulting image reveals a complex structure with five‑fold symmetry.

## 7. Discussion

The proposed generalised logistic map with memory and feedback is a natural extension of the classical logistic map. It introduces two additional parameters (i and k) and a threshold mechanism that can produce both chaotic and stabilised behaviour.

When extended to two dimensions with golden‑ratio scaling and pentagonal rotations, the system generates strange attractors that are visually striking and mathematically rich. These structures may be relevant to the study of quasicrystals, phyllotaxis, and other natural patterns where the golden ratio appears.
